# Infraestrutura como Código (IaC) com Terraform

Repositório dedicado aos meus estudos práticos de **Infraestrutura como Código (IaC)**, com foco na criação, gerenciamento e automação de infraestrutura utilizando **Terraform** em diferentes provedores de nuvem.

O objetivo é consolidar conhecimentos por meio de laboratórios e projetos práticos, aplicando conceitos de automação, provisionamento de infraestrutura, boas práticas e integração com ferramentas do ecossistema DevOps.

## Objetivos

- Aprender os fundamentos de Infraestrutura como Código (IaC).
- Provisionar recursos utilizando Terraform.
- Automatizar a criação e gerenciamento de infraestrutura.
- Integrar Terraform com diferentes provedores de nuvem.
- Aplicar boas práticas de versionamento e organização de projetos.
- Evoluir para cenários mais próximos de ambientes corporativos.

## Tecnologias

- Terraform
- Docker
- AWS
- Azure
- Databricks
- Git e GitHub
- AWS CLI
- Azure CLI

## Objetivo Final

Construir uma base sólida em Infraestrutura como Código utilizando Terraform, preparando o ambiente para projetos reais em cloud e integração com práticas modernas de DevOps e Data Engineering.
